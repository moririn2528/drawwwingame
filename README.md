## 依存の解釈について
内部は外部のことを知らないようにする、と言っているが、実際に書こうとすると、外部を書いているときにこの機能が欲しい→ interface を書くので依存している。
したがって、"依存(A→B)" は、B を取り除いたときに A は実行できず、A を取り除いたとしても B を実行できるものの関係、とした。
